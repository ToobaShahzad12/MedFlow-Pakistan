# MedFlow Pakistan

MedFlow Pakistan is a healthcare queue management and digital token booking UIUX prototype designed to improve hospital usability, accessibility, and patient flow.

## Problem Statement

Patients in hospitals often face long waiting lines, overcrowded reception areas, and confusing token systems. This project was designed to simplify healthcare navigation and improve user experience through a user-centered digital solution.

## Features

 Digital token booking
 Live queue tracking
 UrduEnglish bilingual support
 Interactive onboarding guidance
 Validation-based interaction logic
 Reusable components and variants
 Accessibility-focused UI
